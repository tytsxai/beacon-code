# @tytsxai/beacon-code-responses-api-proxy

<p align="center"><code>npm i -g @tytsxai/beacon-code-responses-api-proxy</code> to install <code>code-responses-api-proxy</code></p>

This package distributes the prebuilt [Beacon Responses API proxy binary](https://github.com/tytsxai/beacon-code/tree/main/code-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/code-responses-api-proxy.js --help
```

Refer to [`code-rs/responses-api-proxy/README.md`](https://github.com/tytsxai/beacon-code/blob/main/code-rs/responses-api-proxy/README.md) for detailed documentation.
